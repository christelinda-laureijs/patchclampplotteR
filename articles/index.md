- [Evoked Current
  Analysis](https://christelinda-laureijs.github.io/patchclampplotteR/articles/evoked-current-analysis.md):

  Learn how to analyze raw evoked current data in Clampfit and prepare

- [Action Potential
  Analysis](https://christelinda-laureijs.github.io/patchclampplotteR/articles/action-potential-analysis.md):

  Learn how to obtain action potential properties from a raw .abf file
  in Clampfit.

- [Spontaneous Current
  Analysis](https://christelinda-laureijs.github.io/patchclampplotteR/articles/spontaneous-current-analysis.md):

  Learn how to analyze spontaneous current data using Event Detection in
  Clampfit, and then prepare the data for use with the package
  patchclampplotteR.
