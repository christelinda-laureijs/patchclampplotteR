# patchclampplotteR: Plot and Analyze Raw Patch Clamp Electrophysiology Data

Create publication-quality figures of whole-cell patch clamp
electrophysiology data. You can plot evoked and spontaneous current
amplitude over time, mean evoked/spontaneous data summarized by sex,
variance analysis plots, and much more.
